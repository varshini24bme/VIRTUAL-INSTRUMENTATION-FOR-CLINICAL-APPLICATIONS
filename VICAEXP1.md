#  Experiment 1 – Basic Arithmetic Operations using LabVIEW

##  Aim
To perform basic arithmetic operations (Addition, Subtraction, Multiplication, Division) using LabVIEW.

---

##  Tools Required
- National Instruments LabVIEW (any version supporting basic VI creation)

---

##  Algorithm

| Step | Description |
|------|-------------|
| 1 | Start LabVIEW and select **Blank VI** |
| 2 | Create the **Front Panel** and **Block Diagram** panel |
| 3 | Add **Numeric Controls** as inputs and **Numeric Indicators** as outputs via right-click on the Front Panel |
| 4 | Place arithmetic operators (`+`, `-`, `×`, `÷`) in the **Block Diagram** panel |
| 5 | Use **wiring** to connect inputs and outputs to their respective operators |
| 6 | Enter input values in the Front Panel and **run** the program to observe output |
